# اپ گزارش روزانه پروژه ساختمانی

نسخه MVP آفلاین برای ۸ زون و ۱۴۱ بلوک.

## امکانات
- رابط فارسی و راست‌چین
- نقش نمایشی سرناظر / مدیر پروژه
- ثبت نیروی انسانی بر اساس زون، بلوک و فعالیت
- ثبت ماشین‌آلات
- داشبورد جمع‌بندی زون‌ها و کل پروژه
- داده‌ها در حافظه برنامه نگهداری می‌شوند (نسخه MVP)

## اجرا
1. Flutter SDK و Android Studio را نصب کنید.
2. در پوشه پروژه اجرا کنید:
   flutter pub get
   flutter run
3. ساخت APK:
   flutter build apk --release

فایل خروجی معمولاً در مسیر زیر ایجاد می‌شود:
build/app/outputs/flutter-apk/app-release.apk

## مرحله تولیدی پیشنهادی
برای استفاده چندکاربره واقعی باید احراز هویت، دیتابیس مرکزی، API، همگام‌سازی آفلاین، آپلود عکس و سطح دسترسی به پروژه افزوده شود.
